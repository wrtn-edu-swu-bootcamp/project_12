# 데이터 분석 시간 예측 서비스

**"쉽고 빠르게, 분석 시간을 미리 알자!"**

데이터 분석 시작 전에 **30초 입력 → 1초 예측**으로 분석 소요 시간을 즉시 확인하는 웹 서비스입니다.

## 🎯 핵심 기능

- ⚡ **빠른 예측**: 30초 입력, 1초 결과
- 📊 **17개 분석 방법 지원**: 집계(3) / 회귀(4) / 분류(4) / 클러스터링(4) / 딥러닝(2)
- 🖥️ **하드웨어 성능 반영**: 4단계 사양 (저사양 ~ 최고사양)
- 📈 **신뢰 구간 표시**: 최소~최대 범위 (±30%)
- 🔍 **단계별 시간 분해**: 로딩 / 전처리 / 분석 시간

## 🚀 빠른 시작

### 1. 웹 서비스 사용

브라우저에서 `index.html` 파일을 열면 즉시 사용 가능합니다.

**필수 입력 항목:**
1. 데이터 행 수 / 열 수
2. 분석 방법 (17개 중 선택)
3. 사용 툴 (Python, R, SQL, Excel)
4. 하드웨어 성능 (필수)
   - 저사양: 2코어, 4GB RAM, HDD
   - 중간: 4코어, 8-16GB RAM, SSD
   - 고사양: 8코어, 32GB RAM, SSD
   - 최고사양: 16코어 이상, 64GB+ RAM, SSD + GPU
5. 데이터 타입 비율 (필수, 합계 100%)
   - 수치형 / 범주형 / 텍스트

### 2. 벤치마크 데이터 수집 (선택)

실제 환경에서 벤치마크 데이터를 수집하여 예측 정확도를 향상시킬 수 있습니다.

```bash
# 필요 패키지 설치
pip install -r requirements.txt

# 벤치마크 데이터 수집
python benchmark_collector.py
```

수집된 데이터는 `benchmark_data.json`에 저장됩니다.

## 📋 지원하는 분석 방법

### 📊 단순 집계 (3개)
- 기본 집계 (평균, 합계, 개수)
- 그룹별 집계 (GROUP BY)
- 피벗 테이블

### 📈 회귀분석 (4개)
- 단순 선형 회귀
- 다중 선형 회귀
- 릿지 회귀 (Ridge)
- 다항 회귀 (Polynomial)

### 🎯 분류 (4개)
- 로지스틱 회귀
- 의사결정나무
- 랜덤 포레스트
- SVM

### 🔍 클러스터링 (4개)
- K-means (k ≤ 10)
- K-means (k > 10)
- DBSCAN
- 계층적 클러스터링

### 🧠 딥러닝 (2개)
- 간단한 신경망 (2-3 레이어)
- 깊은 신경망 (5+ 레이어)

## 📈 사용 예시

### 시나리오 1: 점심시간 활용
- **입력**: 100만 행 × 25열, 랜덤 포레스트, Python, 중간 사양
- **예측**: 45분 (30분 ~ 60분)
- **의사결정**: 점심시간에 실행하기로 결정 ✅

- **예측**: 12분 (8분 ~ 18분)
- **의사결정**: 지금 바로 실행 가능 ✅
